How to use

0. Load the ccm framework

Somewhere in your html:
<script src="http://akless.github.io/ccm-developer/libs/ccm.min.js"></script>

1. Register bugtrackerMwe component to ccm framework

Directly in html for rendering: 
<script>
ccm.render('http://moritzkemp.github.io/bugtracker-mwe/resources/js/ccm.bugtrackerMwe.js', 
    {
        element: $('#bugtrackerWebsiteArea')
    }
);
</script>
or just create an instance for using inside your own ccm-component:
ccm.instance('http://moritzkemp.github.io/bugtracker-mwe/resources/js/ccm.bugtrackerMwe.js', 
    {
        element: $('#bugtrackerWebsiteArea')
    }
);
Refer to the API-documentation for the correct component configuration.
